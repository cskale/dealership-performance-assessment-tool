OEM-Ready SaaS — Feature Landscape

Pick the area(s) you want to focus on first. Each is an independent subsystem.

A

Programme Management

OEMs create named improvement programmes (e.g. "Gold Dealer 2026"), assign target KPIs, set minimum score thresholds per tier, and track dealer compliance across the network. Dealers see their programme requirements on their dashboard.

HIGHEST OEM VALUE MEDIUM EFFORT
B

Delta Scoring / Trend Analysis

Track score changes between assessments. Show dealers their improvement trajectory (↑+8 pts in Service). Show OEMs which dealers are improving fastest vs stagnating. Powers the "ROI of coaching" narrative. Already partially designed in tracker #36.

HIGH VALUE LOW EFFORT
C

Automated Network Reports

Scheduled PDF/email digests sent to OEM programme managers: network health score, top/bottom performers, departments needing attention, coach activity summary. Sent weekly or monthly. Makes the tool "sticky" — OEM stays informed without logging in.

HIGH VALUE HIGH EFFORT
D

Context Intake / Pre-Assessment Data

Before assessment begins, dealer enters actual KPIs (volume sold, CSI score, staff count, etc.). These anchor the diagnostic — questions become calibrated to real numbers, not guesses. Scores become defensible to OEM. Tracker item #12.

HIGH VALUE MEDIUM EFFORT
E

Coach Workflow (Notes + Visit Log)

Coaches log site visits, add private notes per dealer, record action plan sign-offs, and flag blockers. OEM sees coach activity cadence per dealer. Turns the tool from "assessment software" into "field coaching platform."

HIGH VALUE MEDIUM EFFORT
F

White-Label / OEM Branding

Each OEM network sees their own logo, brand colours, and product name (e.g. "BMW Dealer Excellence" instead of "Dealer Diagnostic"). Custom subdomain per OEM. Makes the tool feel like the OEM's own product — critical for enterprise sales.

SALES-CRITICAL HIGH EFFORT

My recommendation: Start with B (Delta Scoring) — lowest effort, immediate visible value for dealers AND OEMs, unblocks the coaching narrative, and is already partially designed in tracker #36. Then A (Programme Management) as the flagship OEM feature.
